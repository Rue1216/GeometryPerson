## Tech Stack

1. 使用 HTML / SCSS / JavaScript 完成頁面切版
2. 分類機制實作
3. 使用 GSAP 進行動態效果製作

## 規則

① 設定選項分數（不一定要按順序）：

- 選項一＝ 1 分
- 選項二＝ 3 分
- 選項三＝ 5 分

② 設定分數級距（不一定要很平均）：

- 3-5 分＝類型一
- 6-11 分＝類型二
- 12-15 分＝類型三
